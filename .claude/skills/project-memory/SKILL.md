# Project Memory System

## Memory Files
- docs/decisions/decisions.md - Architectural decisions
- docs/bugs/bugs.md - Bugs and solutions
- docs/facts.md - Key project facts

## When to Update
- Bug solved → add to bugs.md
- Architecture decision → add to decisions.md
- Non-obvious config/solution → add to facts.md

## Decision Format
## [DATE] - [TITLE]
**Context:** Why needed
**Decision:** What we chose
**Reasoning:** Why
**Alternatives:** What else considered

## Bug Format
## [DATE] - [SYMPTOM]
**Symptom:** What happened
**Root Cause:** Why
**Solution:** Fix
**Prevention:** Avoid in future
